.. _glossary:

Glossary
========

.. glossary::
   Applist
      The applist file describe a list of application packages which will be
      deployed on a workstation or a server. The `user manual`_ details the
      content of this file.

   Catalog
      A file where :class:`AppDownload` class (resp. :mod:`appdownload` script)
      stores information (name, version, published date,...) about the handled
      products.

   Product
      A product is an commercial software tracked and deployed by lAppUpdate.
      It is commercial or a custom software, where commercial term covers
      `commercial off-the-shelf`_ (COTS), free software and open source software
      [#f1]_.

.. rubric:: Footnotes

.. [#f1] Free-Libre / Open Source Software (FLOSS) is Commercial Software,
   http://www.dwheeler.com/essays/commercial-floss.html.


.. _commercial off-the-shelf:  https://en.wikipedia.org/wiki/Commercial_off-the-shelf
.. _user manual: http://fmezou.github.io/lappupdate/lappupdate_wiki.html#applist
